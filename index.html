1 Multiple Cursors

Most editors/ides have vertical column selection.. what makes sublime slightly better is multiple cursors. Multiple cursors illiminate lots of repetative tasks that I used to do in visual studio. Also means I don't have to rememer horrible regular expressions. In this demo I've been asked to extract out of an xml file.. all of the authors and put them into a unordered list. Here's a quick demo I prepared earlier which shows you what I'm talking about

2 Command Pallet

Control Shift P.. can't remember shortcut? who cares.. with the command pallet you don't have to It's extenable.. as you install new packages you'll get new commands..

3 Package manager

Out the box sublime is cool enough but there are lots of handle plugins waiting to be installed First thing you should do is install the package control plugin with this easy to remember python script: here Then it's a case of Control Shift P.. and install.. then you can select from thousands of packages,, themems settings etc.
